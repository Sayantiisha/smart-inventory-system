🤖 AI Powered Smart Inventory Management System

📌 Project Overview

The AI Powered Smart Inventory Management System is a full-stack web application developed using React, FastAPI, SQLite, and Machine Learning. It helps businesses manage products, track inventory, analyze sales, and predict future sales using an AI model.

This project demonstrates full-stack development, REST API integration, database management, data analysis, and machine learning in a single application.

---

✨ Features

- 📦 Product Management (CRUD Operations)
- 📊 Dashboard with Live Statistics
- 📈 Sales Analytics
- 🤖 AI Sales Prediction
- 🔍 Product Search & Filtering
- ⚡ FastAPI REST API
- 🗄️ SQLite Database
- 🎨 React Frontend
- 📉 Data Visualization using Matplotlib & Plotly

🎯 Machine Learning Model

The project trains and compares multiple regression models:

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

The best-performing model is selected for sales prediction.
